# 商业街素材需求表

规格基准：地图 `26 x 17` 格，单格按 `64 x 64 px` 制作。素材统一透明 PNG，像素风，正面或轻微俯视角，不带阴影底色；阴影由前端统一加。

## 命名与交付

| 项 | 规则 |
| --- | --- |
| 文件名 | `building_agency.png`、`prop_notice_board.png` 这种小写蛇形命名 |
| 目录 | 商业街素材放 `client/public/assets/pixel-world/generated-commercial/game-ready/` |
| 背景 | 必须透明 |
| 尺寸 | 可以大于占地格，但画面不要留太多空白 |
| 风格 | 暖色、轻松、商业街像素风，和现有 generated-commercial 素材统一 |
| 交付附带 | 每张图最好给一个简短说明：推荐占地、可碰撞区域、门口/交互点 |

## 摆放硬规则

| 规则 | 说明 |
| --- | --- |
| 主路保留 | `y=8..10` 至少保留 2 格宽连续可走路面 |
| 门口空地 | 每个建筑门口前保留 `2 x 1` 空地 |
| 碰撞不重叠 | 建筑、树、摊位、公告牌的 `solid` 不能互相重叠 |
| 锚点可达 | 所有交互锚点必须能从 `bus_stop` 走到 |
| 大建筑上半区 | 建筑集中放 `y=0..6`，不要压到主路 |
| 小道具贴边 | 小道具贴建筑边、草地边或路边，不能放在人行道中央 |
| 广告牌可读 | 公告牌周围不要堆高物，角色停留时气泡不能被建筑挡住 |
